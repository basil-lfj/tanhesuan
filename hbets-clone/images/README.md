## 图片获取建议

1. **从原始网站下载**：访问 https://www.hbets.cn/ 获取原始图片资源
2. **使用设计工具创建**：使用Photoshop、Figma等工具创建符合主题的图片
3. **使用图库资源**：从免费图库网站（如Unsplash、Pexels）获取合适的图片

## 图片优化建议

- 压缩图片文件大小，提高加载速度
- 使用WebP格式支持现代浏览器
- 为不同屏幕尺寸准备不同分辨率的图片
- 添加适当的alt文本以提高可访问性
