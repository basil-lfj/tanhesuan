# 快速开始指南

## 如何查看网站

### 方法一：直接打开文件
1. 找到 `index.html` 文件
2. 双击文件或右键选择"打开方式" → 选择浏览器（推荐Chrome或Edge）

### 方法二：使用本地服务器（推荐）
使用Python启动简单服务器：

**Windows系统：**
```bash
# 打开命令提示符或PowerShell
cd d:\小挑\hbets-clone
python -m http.server 8000
```

然后在浏览器中访问：http://localhost:8000

## 自定义修改

### 替换Logo
编辑 `index.html` 第26行：
```html
<img src="images/logo.svg" alt="河北电子口岸" class="logo-img">
```

### 替换轮播图
编辑 `index.html` 第52、58、64行的图片URL：
```html
<img src="你的图片URL" alt="Banner 1">
```

### 修改轮播图速度
编辑 `script.js` 第11行：
```javascript
autoPlayInterval = setInterval(nextSlide, 5000); // 5000 = 5秒
```

### 修改主题颜色
编辑 `style.css` 第31行：
```css
background: linear-gradient(135deg, #0052a3 0%, #0066cc 100%);
```

## 常见问题

### Q: 图片显示不出来？
A: 目前使用的是在线占位图服务，如果无法访问，请检查网络连接或替换为本地图片。

### Q: 动画不流畅？
A: 建议使用Chrome或Edge等现代浏览器以获得最佳性能。

### Q: 移动端显示异常？
A: 请确保使用响应式浏览器或移动设备查看项目。

### Q: 如何添加更多轮播图？
1. 在HTML中添加新的 `banner-slide` 元素
2. 在JavaScript中添加对应的 `dot` 元素
3. 调整轮播图索引范围
